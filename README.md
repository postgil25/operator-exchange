# ⚙️ Operator Exchange
**Verified Skill. Negotiated Rates. Zero Friction.**

Operator Exchange is a "headless" directory platform designed to bridge the gap between high-intent employers and skilled construction equipment operators in Nigeria.

## 🚀 Key Features
- **Headless CMS:** Managed entirely via a public Google Sheet. No database required.
- **BACS Scoring:** Proprietary competence algorithm (7.0–9.9) based on seat hours, environment mastery, and technical logic.
- **Video Proof First:** Integrated player for operator-submitted proof-of-skill videos.
- **Intent-Gated Negotiation:** Direct-to-WhatsApp routing for hiring and custom maneuver requests.

## 🛠 Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Modern Glassmorphism), Vanilla JavaScript.
- **Data Layer:** Live CSV ingestion via Fetch API.
- **Deployment:** GitHub Pages.

## 🔧 Setup
1. Clone the repo.
2. Update `CSV_URL` in `app.js` with your published Google Sheet CSV link.
3. Update `WA` constant in `app.js` with your business WhatsApp number.
4. Deploy to GitHub Pages.
